# Competency questions index

- **cq01_tcf7l2_pleiotropy** — Which cardiometabolic SNPs map to TCF7L2 and their effect sizes?
- **cq02_highest_fiber_foods** — Which foods have the highest total dietary fiber content?
- **cq03_butyrate_producers** — Which gut microbes are known to produce butyrate or other short-chain fatty acids?
- **cq04_top_cardiometabolic_genes** — Which genes carry the most cardiometabolic GWAS signals?
- **cq05_phenotype_coverage** — How many SNPs are associated with each cardiometabolic phenotype?
- **cq06_foods_via_microbe_nutrient_obesity** — Which foods contain nutrients also produced by obesity-associated microbes?
- **cq07_microbes_modulating_gwas_genes** — Which gut microbes modulate host genes that also carry cardiometabolic GWAS variants?
- **cq08_t2d_associated_microbes** — Which gut microbes are directly associated with type 2 diabetes?
- **cq09_food_microbe_shared_metabolites** — Which metabolites are produced by both foods and gut microbes?
- **cq10_tcf7l2_pleiotropy_extended** — How many distinct cardiometabolic phenotypes does each TCF7L2 SNP touch?
- **cq11_full_five_hop_path** — Is there a full Food -> Nutrient <- Microbe -> Gene <- SNP -> Phenotype path? (tri-layer)
- **cq12_pparg_tri_layer_convergence** — Which microbes and SNPs converge on PPARG — a drug target for cardiometabolic disease?
- **cq13_bridge_genes_across_layers** — Which host genes are supported by both gut-microbe modulation AND cardiometabolic GWAS?
- **cq14_scfa_producers_to_gwas_genes** — Do SCFA-producing microbes reach obesity/T2D via host-gene modulation?
- **cq15_evidence_density_by_phenotype** — Which cardiometabolic phenotype has the densest tri-layer evidence?
